# 머신러닝 특성 공학 파이프라인 과제

Titanic 공개 데이터셋을 사용해 특성 공학 파이프라인을 구현한 소스코드 저장소입니다.

## Files

- `feature_engineering_assignment.py`: 데이터 다운로드, EDA, 특성 공학, 모델 비교 실험, 결과 파일 생성

## Run

```powershell
python feature_engineering_assignment.py
```

## Output

- `assignment_outputs/experiment_results.csv`
- `assignment_outputs/result_summary_table.csv`
- `assignment_outputs/feature_importance_details.csv`
- `assignment_outputs/figures/*.png`

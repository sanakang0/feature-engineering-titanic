# Feature Engineering Assignment

Titanic 데이터를 사용한 특성 공학 파이프라인 과제입니다.

## Files
- `feature_engineering_assignment.py`: 데이터 수집, EDA, 특성 공학, 모델 학습, 결과 생성
- `build_assignment_pdf.py`: 제출용 PDF 생성 스크립트
- `과제_특성공학_보고서.md`: 보고서 원본
- `과제_특성공학_보고서.pdf`: 최종 제출 PDF

## Run
```powershell
python feature_engineering_assignment.py
python build_assignment_pdf.py
```
